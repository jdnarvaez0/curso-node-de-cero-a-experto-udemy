# 03 - Bases de Node + TypeScript

Código de la sección 3 del curso: JavaScript avanzado + Loggers (Winston) + Migración a TypeScript.

## 📦 Dependencias

```bash
npm install
```

## ▶️ Ejecutar

```bash
npm run dev        # nodemon ejecuta src/app.js
```

Cuando llegues a la parte de Winston:

```bash
npm install winston
```

Y cuando arranques TypeScript:

```bash
npm install -D typescript @types/node ts-node
npx tsc --init
```
